Project explained:

• Cakephp 2.0 Framework.
• Virtual library with flipbook.
• MySQL Database.
• System Auth with Facebook integration and different security levels (guest, normal user, payed user, administration).
• Paypal integration.
• Uses Plupload, Fancybox, jQuery.
